# Frontend

The React/Vite client for Secure Notes with Expiring Share Links.

Run it with:

```bash
npm install
npm run dev
```

It expects the API at `http://localhost:5000/api` by default. Set `VITE_API_URL` to use another API URL. See the [root README](../README.md) for full setup, share-link behavior, security details, and demo requirements.
